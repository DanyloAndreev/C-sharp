Консольный калькулятор

1. Функции калькулятора:
- (+ - * /), корректно выполняет все операции
- работает с дробными числами
- продолжает работу при вводе неправильных значений
- закрывается при вводе клавиши q
- умеет считать длинные примеры(100+5*9/2-44)
- умеет считать примеры со скобками
- чтобы выдавало ответ буквами и прочитывало его вслух(в версии 5.0)
- конвертация доллара(вводишь $25 потом вводишь курс и получаешь ответ)
- функция рандом(вводишь число, получаешь рандомное число меньше либо такое же число больше ноля)
- те же функции приложение для андроид в версии 5.0
